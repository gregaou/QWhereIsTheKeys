Projet: QWhereIsTheKey's++
Prototype réseau Client.
Création du fichier: 19/11/12
Derniere mise à jour: 19/11/12



Utilisation:
1) Lancer le prototype client.
2) Entrez l'adresse ip du serveur (si le serveur et sur la même machine que le client, "localhost" fonctionne aussi).
3) Entrez le port du serveur sur lequel se connecter (par defaut, le port du prototype serveur est "1234").
4) Facultatif, vous pouvez entrer votre pseudonyme, si vous l'entrez, il apparaitra devant les messages que vous enverrez plus tard.
5) Appuyez sur le bouton "connexion", si le serveur est lancé et que les informations entrées sont correcte, la connexion devrait se faire correctement, autrement le logiciel
essaie de se connecter pendant plusieurs secondes puis re-autorise à re-entrer les informations.
6) Une fois connecté, il est possible d'écrire des messages en bas de l'écran, ces messages seront envoyé au serveur qui les enverra à tous les clients qui lui sont connectés.
7) Apres avoir envoyé le necessaire ou si l'on souhaite changer de serveur, il est possible de se deconnecter du serveur actuel grace au bouton correspendant.
